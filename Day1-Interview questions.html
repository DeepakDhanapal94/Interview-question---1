Ques 1. Difference between div and span?

Div tag -

  1. This will be used as a single line element and its always starts in new line.
  2. It is also used for block-level organization and styling of page elements.
  3. It is a division tag occupied with 100% width.

Span tag-

  1. This will work in same line and can be used within the paragraph or content.
  2. Span element is used for in-line element organization and styling.
  3. It is a separation tag occupies only specific text or content width
